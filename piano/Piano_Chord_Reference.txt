Piano Application Chord Dictionary & Music Theory Reference

OVERVIEW
--------
This document lists all supported chords in the Piano application with their
musical structure, intervals, notes, and theory explanations.

NOTATION SYSTEM
---------------
• Root notes: A, B, C, D, E, F, G (H is accepted as alias for B)
• Sharps: # (raises note by semitone)
• Flats: b (lowers note by semitone)
• Octave range: C4 to B5 (2 octaves on piano display)
• Mode: Triad (3 notes) or Tetrad (4 notes)

INTERVAL NOTATION
-----------------
R    = Root (1st note)
m2   = Minor 2nd (1 semitone)
M2   = Major 2nd (2 semitones)
m3   = Minor 3rd (3 semitones)
M3   = Major 3rd (4 semitones)
P4   = Perfect 4th (5 semitones)
P5   = Perfect 5th (7 semitones)
m6   = Minor 6th (8 semitones)
M6   = Major 6th (9 semitones)
m7   = Minor 7th (10 semitones)
M7   = Major 7th (11 semitones)

TECHNICAL NOTES
================================================================================

ENHARMONIC SPELLINGS
--------------------
The application automatically converts flat notes to sharp equivalents to match
the piano keyboard's visual representation:
  Eb → D#    Ab → G#    Bb → A#
  Db → C#    Gb → F#
  Cb → B     Fb → E
  E# → F     B# → C

TRIAD vs TETRAD MODE
--------------------
• Triad: 3-note chord (root, 3rd, 5th)
• Tetrad: 4-note chord (adds 7th or equivalent 4th note)
• Some chords have identical triad/tetrad structures
• Switch mode using R3 panel buttons: B8 (Triad) / B9 (Tetrad)

INVERSIONS (NOT YET IMPLEMENTED)
---------------------------------
Inversions reorder chord notes:
• Root position: Root is lowest note
• 1st inversion: 3rd is lowest note
• 2nd inversion: 5th is lowest note
• 3rd inversion: 7th is lowest note (tetrads only)

CHORD VALIDATION
----------------
Valid chord format in R1 panel:
  [A-GH][#|b]?[maj|m|dim|aug|sus]?[2|4|5|6|7|9|11|13]?

Examples:
  C, Dm, G7, Cmaj7, Esus4, F#m7, Bmm, Asus2

Invalid examples:
  H#m (H doesn't have sharps), C8, Xm7, C#b
