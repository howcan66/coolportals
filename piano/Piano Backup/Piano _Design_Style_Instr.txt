Piano Design Style Instr

Plain-language guide to the colors, sizes, and styling choices from the referenced piano chord page. Layout is not prescriptive; reuse these tokens in similar pages.

Palette
- Page background: #1e1e1e
- Panels: #2b2b2b
- Borders/dividers: #444; inner headers: #333
- Primary (OK): #0078d4; Secondary/load: #00a86b; Danger/reset: #a80000
- Text: white; secondary: #ddd; tertiary: #ccc
- Piano keys: white #fdfdfd with border #333; black keys solid black; "marked" state: #4da3ff

Typography
- Base font: Arial, sans-serif; body text color white; default size ~1rem
- Section titles/labels: 1.1rem, bold, centered
- Help/footer text: 0.8rem; help line-height 1.3
- Piano key labels: 0.75rem, centered on each key
- Textarea: monospace, 0.9rem

Layout and Spacing
- Page: full-height grid with narrow sidebar (15%) and wide main area (85%); no outer margin/padding
- Sidebar: column layout, 10px padding, 10px gaps, right border 2px solid #444
- Top-right panel: 10px padding, 6px vertical gaps, bottom border 2px solid #444
- Buttons: 4px 8px padding, 4px radius; load/reset-all buttons use 6px padding with 10px then 5px top margins
- Inputs: 4px padding, 1px solid #555, 4px radius; full-width within row
- Piano frames: background #2b2b2b, 1px #444 border, top padding 5px; label bar full width with 5px padding
- Piano key area: 300px width, 150px height; black keys 24px wide, 90px tall, absolutely positioned with small bottom radius

States
- Buttons force white text
- Marked keys (white/black) use highlight blue #4da3ff
- Textarea: white background, dark text, 1px #555 border, 4px radius; resizable vertically

Use this sheet to replicate the same visual tone (colors, sizes, typographic choices) even if components or layout differ.
