=============================================================================================================================
Lauout Instruction for AI
=============================================================================================================================
Läs in hela instruktionstexten nedan och fortsätt först när allt är inläst.
När instruktionstexten är helt inläst ska du följa varje steg i ordning.

Rita en horisontell ram över hela bredden.
Placera ordet HEADER centrerat på nästa rad.
Rita en ny horisontell ram över hela bredden.
Dela layouten vertikalt i två områden: en smal vänsterdel och en bred högerdel.
Rita en obruten vertikal rektangel i vänsterdelen som sträcker sig från direkt under den övre ramen till bottenramen.
Placera texten R1 högst upp i vänsterrektangeln.
Till höger om vänsterrektangeln, rita en horisontell linje som markerar toppen av R2-området.
Dela R2-området i två lika breda vertikala sektioner med en lodrät linje.
Placera texten R2A i den vänstra sektionen av R2-området.
Placera texten R2B i den högra sektionen av R2-området.
Förläng höjden på båda sektionerna så att R2-området blir flera rader högt.
Rita en horisontell linje under R2-området, utan att korsa vänsterrektangeln.
Under denna linje, skapa en stor rektangel som täcker hela högerdelen — detta är R3-området.
Placera texten R3 i den övre delen av denna rektangel.
Lämna flera tomma rader i R3-området för att matcha den totala höjden i layouten.
Rita en bottenram som avslutar hela ASCII-strukturen.

Nedan är det färväntade resultatet

+-----------------------------------------------------------------------------------+
|                                      HEADER                                       |
+-----------------------------------------------------------------------------------+
| R1                  |-----------------------------------------+-------------------|
|                     |                  R2A                     |        R2B       |
|                     |                                          |                  |
|                     |                                          |                  |
|                     |                                          |                  |
|                     |                                          |                  |
|                     |------------------------------------------+------------------|
|                     |                  R3                                         |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
|                     |                                                             |
+-----------------------------------------------------------------------------------+

När alla steg är utförda ska du som sista moment skriva ut den färdiga ASCII-layouten.
